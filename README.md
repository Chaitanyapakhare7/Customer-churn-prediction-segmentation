# Churn Prediction and Customer Segmentation

A machine learning project to predict customer churn and segment customers based on behavioral patterns. Built as part of the CBSOT SIP (AI/ML/Data Science) internship program.

## Objective

Predict whether a customer is likely to churn and segment the customer base into distinct groups to support targeted retention strategies.

## Dataset

Telco Customer Churn Dataset — contains customer demographics, account information, subscribed services, and churn status.

## Approach

1. **Exploratory Data Analysis** — examined tenure, monthly charges, and contract type in relation to churn
2. **Data Preprocessing** — label encoding, handling missing values
3. **Model Training** — Random Forest Classifier for churn prediction
4. **Customer Segmentation** — KMeans clustering on customer behavior
5. **Insights** — identified key churn drivers and high-risk customer segments

## Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## How to Run

1. Clone this repository
2. Open `Churn_Prediction_Segmentation.ipynb` in Jupyter Notebook or Google Colab
3. Run all cells in order
